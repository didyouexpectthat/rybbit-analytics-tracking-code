Add the Rybbit tracking code to your WordPress site.

== Description ==

This community-maintained plugin adds Rybbit tracking code to your WP site in a few easy steps.

== Disclaimer ==

Rybbit is a trademark and copyright of Rybbit. This community-maintained plugin for WordPress is endorsed on [Rybbit's website](https://www.rybbit.io/docs/integrations/wordpress). 

= Features =

* Simple configuration through WordPress admin
* Automatic script insertion in site header

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/rybbit-analytics-tracking-code` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings -> Rybbit screen to configure the plugin

== Frequently Asked Questions ==

= What is Rybbit? =

Rybbit is a tracking and analytics service. You can learn more about Rybbit at https://rybbit.io

= What is Rybbit Analytics Tracking Code? =

This plugin helps you integrate it with your WordPress site quickly and easily. Add your tracking URL, your site ID, and customize as needed.

= Do I need a Rybbit account to use this plugin? =

As the WordPress administrator, you need to know the URL to use and the site ID given by the Rybbit administrator or interface.
